Spaceball
---------------------
0x100 0xXX, Y - Input
X = Time in Ticks (Used Values: 0x30, 0x60)
Y = Object Type
0 - Baseball
1 - Rice ball
2 - Alien

0x101 X - Costume
0 - Normal
1 - Red Head
2 - Bunny

0x102 Y, Z - Camera
X = Interpolation
0 - Instant Zoom
1 - Linear Zoom
2 - Smooth Zoom
Y = Predetermined Position (0~0xB, 4 is default)
Z = Time in ticks (Used Values: 0x18, 0x30, 0x60, 0x90, 0xC0, 0x120, 0x180, 0x240, 0x2A0)

0x103 - Alien
0 - Opening Animation
1 - Closing Animation
2 - Gone

0x104 X - Unused Animation
0 - dance00
1 - dance01
