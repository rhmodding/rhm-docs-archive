The Clappy Trio
---------------------
0x100 X - Input
X = Interval between claps in ticks (Used Values: 6, 0xC, 0x18, 0x24, 0x30, 0x60)
The input happens after twice as many ticks as X have passed since 0x100 X

0x101 X - Beat Animation
0 - Normal
1 - Normal (Usually used 1.0 Beats Before Ready Stance)
0x101<1> X - X is usually set to the amount of beats until the ready stance (No Effect)

0x102 - Ready Stance
0 - Normal
1 - Determined

0x103 Y - Clap Animation
X = Type
0 - Normal
1 - Normal (Unused)
Y = Which One
0 - Left Guy
1 - Middle Guy
2 - You (No Visible Effect)

0x104 X - Beat Animation Type
0 - Neutral Faces (Unused)
1 - Happy Faces

0x105 0xYYYYYY - Sign
X = Action
0 - Sign drops down
1 - Sign disappears
2 - Sign is already down
Y = String/Layout (Trio Timing!)
0x503074 - "Clap_ver_arrange.bflyt" Trio Timing (The Snappy Trio)
0x503122 - "Clap_ver_long.bflyt" Trio Timing! (The Clappy Trio 2)
0x5031CD - "Clap_ver_short.bflyt" Trio Timing! (The Clappy Trio)
0x509CDB - "Clap_ver_remix.bflyt" Trio Timing (Citrus Remix)
