Sneaky Spirits
---------------------
0x100 X - Input (After 1.0 Beat from the command)
X = Type
0 - Normal
1 - ???
2 - ???
3 - Practice (Doesn't laugh when you miss)

0x101 - Draw Bow
0x101<1> X, Y - Slide Bow into View
X = Starting Position (Used Values: 0, 0xFA)
Y = Time in Ticks (Used Values: 0, 0x60)

0x102 X - Spirit Pops Up
X = Position (0 ~ 6)
0x102<1> - Pause ghost mid-jump with arrow (Practice)
0x102<2> - Unpause ghost

0x103 - Close Door

0x104 X - Spirit Height
X = Height (0 ~ 0x100)
0x104<1> X - Music Goes Quiet when Spirits go Lower
0 - Disable (Remix)
1 - Enable (Default)

0x105 X - Spirit Position
0 - Hit Zone (Temporarily stops ambience noise if hit)
1 - Behind the Fence
0x105<1> X - ???
0 - ??? (Unused?)
1 - ???

0x106 X - Butterflies
X = Amount (Used Value: 6)

0x107 - Reset Speed
0x107<1> X - Game Slows Down when hitting a Spirit
0 - Disable (Remix)
1 - Enable (Default)
