Marching Orders
---------------------
0x100 X, Y - Input
X = Time in Ticks (Used Values: 0xC, 0x30)
Y = Action
0 - March
1 - Halt
2 - Right-Face Turn
3 - Left-Face Turn

0x101 - Raise Legs

0x102 - Beat Animation

0x103 - Reset Stance

0x104 - March Animation
0x105 - Halt Animation
0x106 - Right-Face Animation
0x107 - Left-Face Animation
X = Which One (1, 2, 3 are used, in order from right to left)

0x108 - Sarge Talk

0x109 - Conveyor
---------------------
Subs:
0x56 - "Turn!" (Right-Face)
0x57 - "Turn!" (Left-Face)
0x58 - "March!"
0x59 - "Halt!"
0x5A - Marching Orders (Practice)
0x5B - Marching Orders (Real Game)
0x5C - Marching Orders (Skipped Practice)
