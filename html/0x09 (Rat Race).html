Rat Race
---------------------
0x100 X, Y - Input
X = Type
0 - Hold A
1 - Release A
Y = Time in Ticks (Used values: 0x30, 0x90)

0x101 X - Background Movement
0 - Disable
1 - Enable

0x102 X - Rat Animations
0 - Hiding
1 - Running
2 - Preparing to Run
3 - Running Start

0x103 - Rats Celebrate (And Stop Background Movement)

0x104 X - Cat Animations
0 - Left Paw
1 - Right Paw
2 - Head Pops Up
3 - Head Pops Down
4 - Look Straight
5 - Look Left
6 - Look Right
7 - Close Eyes
8 - Paws Back Down

0x105 X - Speed
0 - None
1 - Normal
2 - Fast

0x106 X, Y - Spawn Foreground Object
X = Time in Ticks (Used Value: 0xC0)
Y = Type
0 - Plates/Donuts
1 - Cheese

0x107 - Stoplight (Pull Out)
0x107<1> - Stoplight (Put Away)
0x107<2> X - Stoplight (Yellow)
0 - Left
1 - Middle
2 - Right (Unused)
3+- None (Makes a Sound Anyway)
0x107<3> - Stoplight (Red)
0x107<4> - Stoplight (Green)
0x107<5> - Stoplight (Off)
0x107<6> - "No more signals!"

0x108 - Stop Spawning Background Objects
---------------------
Subs:
0x56 - Stop
0x57 - Go
0x58 - Stoplight (for Stop)
0x59 - Stoplight (for Go)
0x5A - Stoplight (for Go, throw away stoplight) (Unused)
0x5B - Rat Race (Practice)
0x5C - Rat Race (Real Game)
0x5D - Rat Race (Skipped Practice)
