Ninja Bodyguard
---------------------
0x100 X - Input
X = Time in Ticks (Used value: 0x180)

0x101 - Scene
0 - Left (Enemies)
1 - Middle (Intro)
2 - Right (Ninja)

0x102 X - Enemy
X - Amount (up to 6 on screen)
0x102<1> X - Draw Bow (-1 effects all)
0x102<2> X - Shoot Arrow
X - Which one

0x103 X - Ninja
0 - Disable
1 - Enable (Unused)

0x104 - Flying Arrow (Intro)
0x104<1> - Arrow lands on wall
0x104<2> - Ninja jumps in
0x104<3> - Ninja prepares sword
0x104<4> X - Lord walks to position in X ticks (Used value: 0x1F8)
0x104<5> - Lord faces forward
0x104<6> - Ninja jumps away (Practice)

0x105 - Practice Ninjas, Wooden Dummy
0x105<1> - Enemy Ninjas, Lord (Unused)
0x105<2> X - Enabled in Practice (No Visible Effect)
0 - Disable
1 - Enable
0x105<3> - Sets condvar to 1 if player skipped the practice/intro

0x106 - Monkey (Final Remix)
0 - Flies in
1 - Beat Animation
