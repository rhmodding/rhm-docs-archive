Lockstep
---------------------
0x100 X, Y - Input
X = Type
0 - Offbeat
1 - Onbeat
Y = Time in Ticks (Used Values: 0x18, 0x30)

0x101 - Beat Animation

0x102 - Step

0x103 X - Direction
0 - Offbeat
1 - Onbeat

0x104 X - Zoom
0 - Normal
1 - Zoom Out (1)
2 - Zoom Out (2)
3 - Zoom Out (3)
4 - Zoom Out (Portrait)
5 - Practice

0x105 X - Background Color
0, 2, 4, 6 - Light
1, 3, 5, 7 - Dark
---------------------
Subs: (sub5A~sub61 use the second argument, 0x30)
0x56 - March (Onbeat, 4.0 Beats)
0x57 - March (Onbeat, 4.0 Beats) (Transition)
0x58 - March (Offbeat, 4.0 Beats)
0x59 - March (Offbeat, 2.0 Beats) (Transition)
0x5A - Normal Zoom
0x5B - Zoom Out (1)
0x5C - Zoom Out (2)
0x5D - Zoom Out (3)
0x5E - Zoom Out (Portrait)
0x5F - Practice Zoom
0x60 - Background (On to Off)
0x61 - Background (Off to On)
0x62 - Voice Clip ("Hai!")
0x63 - Voice Clip ("Hai, Hai, Hai, Hahai!")
0x64 - Voice Clip ("Ho, Ho, Ho, Ho")
0x65 - Voice Clip ("Nha, Nha")
0x66 - Voice Clip ("Ho, Ho, Nha, Nha")
0x67 - Lockstep (Practice)
0x68 - Lockstep (Real Game)
0x69 - Lockstep (Skipped Practice)
