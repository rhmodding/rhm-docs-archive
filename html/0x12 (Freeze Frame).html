Freeze Frame
---------------------
0x100 X, Y - Input
X = Slot Number of Car (0, 1, 2)
Y = Time in Ticks (Used Value: 0x60)

0x101 X - Reticle
0 - Disable
1 - Enable

0x102 X - Camera Overlay
0 - Disable
1 - Enable

0x103 V, W - Car
0x103 V, X, Y, Z - Move Car
U = Placement
1 - Background
2 - Foreground
V = Slot Number of Car
W = Type
0 - Yellow
1 - Red
X = X Position (Start) (Used Values: -0xBB8, 0xEC, 0xF0, 0x104, 0x11C, 0x134)
Y = X Position (End) (Used Values: -0x11C, -0x104, -0xF0, -0xEC, -0xD4, 0, 0xBB8)
Z = Time in Ticks (Used Values: 0x28, 0x30, 0x60, 0xA0, 0xC0)
0x103<3> 0 - Used in Practice

0x104 - Record Performance
0x104<1> X - Thumbs Up
0 - Disable
1 - Enable
0x104<2> - Sets condvar to player performance
0 - Ace!
1 - Miss
2 - Through

0x105 X, Y - Photos
X = Slot Number of Car
Y = Show
0 - Disable
1 - Enable

0x106 X - Crowd
0 - Disable
1 - Enable

0x107 - Signal (Rhythm Kart)
0x107<1> - Remove Signal
0x107<2> X - Turn on Light
0 - light_00 (Unused)
1 - light_01
2 - light_02
3 - light_03

0x108 X, Y - Walker
X = Priority Slot (0, 1)
Y = Type
0 - Man (White) (Left)
1 - Man (Blue) (Left) (Unused)
2 - Girlfriend (Left)
3 - Man (White) (Right) (Unused)
4 - Man (Blue) (Right)
5 - Girlfriend (Right) (Unused)
