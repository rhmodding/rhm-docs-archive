Glee Club
---------------------
0x100 X - Input (Expected 1.0 Beats after)
0 - Release A (Sing)
1 - Release A (Wail)
2 - Hold A (Stop)

0x101 - Player Hold Check (Used for the "close your yap!" part)
0 - Check if player is holding A
1 - Sets condvar to 1 if holding A
2 - Other members will glare if you don't hold A

0x102 X - Set base pitch (Used value: 0)

0x103 X - Set input delay in X ticks (Unused)

0x104 X, Y, 0xZZZZZZZ, O, P - Voice Control (Set SFX)
0x104<1> X, Y, O - Perform Action
0x104<2> X = Crouch (Before Wailing)
X = Which one (0~2)
Y = Type
0 - Singing
1 - Wailing
2 - Stop
0xZZZZZZZ = SFX
0x1000DAF - Singing
0x1000DB0 - Wailing
0x1000DB1 - Stop
O = Unknown (Used Values: 0, 0x28, 0x2D, 0x3C, 0x46, 0x64, 0x5A, 0x78)
P = Pitch
-8 - C (Unused)
-7 - C#/Db (Unused)
-6 - D
-5 - Eb/D#
-4 - E (Unused)
-3 - F (Unused)
-2 - F#/Gb
-1 - G
0 - Ab/G#
1 - A
2 - Bb/A#
3 - B
4 - C+
5 - C#+/Db+
6 - D+
7 - Eb+/D#+ (Unused)
8 - E+ (Unused)
9 - F+ (Unused)
0xA - F#+/Gb+

0x105 - Conductor
0 - Idle Animation
1 - Raise Baton
2 - Snap Baton

0x106 - Heart

0x107 X, Y - Remove Chorus Kids (Machine Remix)
X = Which one (0~2)
Y = Visible
0 - No
1 - Yes

0x108 - Sets condvar to 1 if X is 1 (Used values: 2)

0x109 - React to the player with an open yap (Animation only)
0 - Disable
1 - Enable

0x10A X - Wailing
0 - Disabled
1 - Enabled

0x10B - Sets condvar to 1 if holding A

0x10C X, Y - Setup something with the rhythm help display? (Unused)
X = Column amount
Y = Row amount

0x10D - Force the game to think that the player had a successful Release A input? (Animation only, used in remixes)
(will still make the other kids give the player the Look if they press A not on input?)
