Fan Club
---------------------
0x100 - Input
0 - Press A (0.5 beats delay)
1 - Hold A (0.5 beats delay)
2 - Release A (1.0 beats delay)

0x101 Y - Monkey Beat Animation
0x102 Y - Monkey Ready for Clapping
0x102<2> - Monkeys Shake ("Ohh...")
0x103 Y - Monkey Clap
0x104 Y - Monkey Free (Stop Clapping)
0x105 Y - Monkey Charge
0x106 Y - Monkey Jump
X = Which Monkey
0 - Specific one (uses Y argument)
1 - All (Doesn't use Y argument)
Y = Which Monkey (Used: 0~0xA, 0xC)

0x107 X - Pop Singer "Beat"
0x108 X - Pop Singer "Peace" ("Yeah, yeah, yeah!")
0x109 Y, X - Pop Singer "Wink" (Twirls Hand), Y = 1 adds stars
0x10A X - Pop Singer "Clap"
0x10B X - Pop Singer "Squat"
0x10C X - Pop Singer "Jump"
0x10D X - Pop Singer "Land"
0 - Pop Singer
1 - Left Backup
2 - Right Backup

0x10E - Pop Singer "Call" ("I suppose!")
0x10E<1> - Pop Singer "BigCall" (Last "I suppose!")

0x10F - Pop Singer "Response" ("I suppose!")

0x110 X, Y, Z - Backup Movements (Fan Club 2)
X = Which Singer
0 - Pop Singer
1 - Left Backup
2 - Right Backup
Y = Distance in units (Used Value: 0xA)
Z = Animation
0 - Walk A
1 - Walk B

0x111 X, Y - Singer Visibility
X = Visible
0 - No
1 - Yes
Y = Which Singer
0 - Pop Singer
1 - Left Backup
2 - Right Backup

0x112 - Flash the Lights (Fan Club)
0x112<1> X - Flash the Stars (Fan Club 2)
0 - Stop
1 - Start (also makes the monkeys hold glowsticks)
0x112<2> - Blue and Yellow Lights (Fan Club 2)
0x112<3> - Pink and Green Lights (Fan Club 2)

0x113 - Record inputs (Meant for if you don't clap during the Applause)
0x113<1> X - Monkey looks at you if you don't clap after X beats (Used Value: 6)

0x114 X - Background
0 - Normal
1 - Darker

0x115 - Select random lyrics for the Practice (sets the conditional variable to 0~4)

0x116 - Confetti Poppers (Fan Club 2)

0x65 - Fan Club 2 (Gameplay)

0x67 - Fan Club 2 (Skipped Practice)
