Big Rock Finish
---------------------
0x100 X, Y - Input
X = Time in Ticks (Used value: 0x30)
Y = Type
0 - Shot 0
1 - Shot 1 (Jump)

0x101 X - Which Strum (Practice)
Used values: 0, 1, 2, 3

0x102 X, Y - Guitar Sound Length in Ticks
X = Shot 0 (Used values: 0xC, 0x10, 0x18)
Y = Shot 1 (Jump) (Used values: 0x60, 0x78, 0xC0)
0x102<1> X - Set Pitch
X = Pitch (Used values: -1, 0, 1, 2, 3, 4, 6, 7)

0x103 X - Light Effect
0 - Base
1 - Green (A, D, G)
2 - Blue (B, E, H)
3 - Red (C, F)
4 - Shot Light (Strumming)

0x104 - Ecto Animations (Player)
0x105 - Boo-Boo Animations (Other Guitarist)
0 - Beat
1 - Touch
2 - Return
3 - Shot 0
4 - Shot 1 (Jump)

0x106 - Spooky Animations (Drummer)
0 - Bass Drum
1 - Snare
2 - Hi-Hat
3 - Cymbal
4 - Tom (Right)
5 - Tom (Left)

0x107 - Audience Animations
0 - Beat
1 - Ready
2 - Used with cheers
3 - Sets condvar to 0 if great, 1 if ok, 2 if miss
5 - Disable (Practice)
6 - Enable (Real Game)
0x107<4> X - Performance Reaction
0 - Great (High Cheer)
1 - Ready (Low Cheer)
2 - Miss (Boo)

0x108 - Freestyle Strums (Practice)
0x108<1> X - Sets condvar to 1 if player has strummed X+1 times (Used value: 4)

0x109 X - Unknown (Used with Skill Star pattern)
0 - Disable
1 - Enable

0x10A - Stop Audience Animation (Used when Booing)
