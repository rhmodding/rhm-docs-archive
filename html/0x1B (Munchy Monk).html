Munchy Monk
---------------------
0x100 X - Input
0 - First (1.0 Beats)
1 - Second (1.5 Beats)
2 - Third (0.5 Beats)
3 - Unused (0.0 Beats)

0x101 - Beat Animation

0x102 X - Dumpling (More than two at a time crashes the game)
0 - White (One)
1 - Pink (Two)
2 - Green (Three)
Everything beyond these is unused but potentially useful:
https://bit.ly/2IHYVL8

0x103 X - Hand
0 - One
1 - Two
2 - Three
0x103<1> - Remove Hand

0x104 - Monk Slides Forward
0x104<1> - Monk Snaps to Position

0x105 - Face Turn (Leftover) (by default uses "face_beat" animation if a "face_turn" animation isn't present in snack.bccad)

0x106 - Monk's Hands Slide Forward
0x106<1> - Monk's Hands Snap to Position

0x107 X, Y - Background Movement
X = Movement
0 - Disabled
1 - Enabled
Y = Time in Ticks until moving at full speed (Used values: 0xC0)

0x108 - Cloud (Final Remix)
0x108<1> A, B, C, D, F - Cloud (Wario...Where?)
A = Starting X Position (Used values: -0x50, -0xF0)
B = Ending X Position (Used values: -0x50, 0xF0)
C = Y Position (Used values: -0x46)
D = Time in Ticks (Used values: 0x30, 0xC0)
F = Destination
0 - Continues offscreen
1 - Stops at B's position
0x108<2> - Cloud (Beat Animation)
