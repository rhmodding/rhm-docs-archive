Air Rally
---------------------
0x100 - Input

0x101 Y, Z - Plane Position
X = Who
0 - Baxter
1 - Forthington
Y = Position
0 - Behind Camera
1 - Baxter's Position
2 - Too Near
3 - Near
4 - Far
5 - Too Far
6+- Glitchy
Z = Interpolation
0 - Smooth
1 - Instant

0x102 - Respawn Birdie

0x103 - Hit Birdie

0x104 - Catch Birdie

0x105 - Used with Catch Birdie

0x106 - Unknown
0x106<1> - Unknown
0x106<2> X, Y, Z - Unknown
0x106<3> X, Y, Z - Unknown
X = ???
Used Values: 0x14, 0x1E, 0x28, 0x32, 0x3C, 0x46, 0x50, 0x5A
Y = ???
Used Values: 0x28, 0x32, 0x46, 0x64, 0x50, 0x96
Z = ???
Used Values: 0x180, 0x300, 0x480, 0x600, 0x900

0x107 - Cloud Speed (Used Values: 6, 0x14, 0x50, 0x64, 0xC8)

0x108 X - Background Object Speed (Takes Effect on Next 0x107) (Used Value: 7)

0x109 R, G, B - Set Color Overlay
X = Type
0 - Background Objects
1 - Clouds
2 - Players
3 - Background
R, G, B - Colors
Used Values:
0xFF, 0xFF, 0xFF - Day (Everything)
0xFF, 0x80, 0 - Sunset (Background Objects, Background)
0xFF, 0xC0, 0x80 - Sunset (Clouds)
0, 0, 0 - Sunset (Players)
0xF0, 0xC8, 0xC8 - Morning (Players, Unused)
0, 0x1E, 0x50 - Night (Background Objects)
0, 0xC0, 0xFF - Night (Clouds)
0xC8, 0xFA, 0xFA - Night (Players)
0xD2, 0xFF, 0xB4 - Night (Background)

0x10A - Apply Color Overlay over X ticks (Used Value: 0x12C)

0x10B X, Y - Arc Time in Ticks
X = From Forthington to Baxter (Used Values: 0x30, 0x60)
Y = From Baxter to Forthington (Used Values: 0x30, 0x60)

0x10C Y - Practice Animations
X = Who
0 - Baxter
1 - Forthington
Y = Animation
0 - Look Forward
1 - Look Away (Unused)

0x10D - ??? (something about "s_you_talk" and "s_com_talk") (Unused)

0x10E X - Forthingthon Animations
0 - Baxter Transforms (Unused)
1 - Mouth Movement (Loop)
2 - Mouth Movement (Single)

0x10F - Background Buildings Lights at Night
0 - Disable
1 - Enable

0x110 - Ambient Noises
0 - Disable
1 - Enable

0x111 - ??? (Unused)

0x112 - Forthington Voice Clip
0 - "Ba bum bum bum"
1 - "1!"
2 - "2!"
3 - "3!"
4 - "4!"

0x113 X - ??? (Unused)
0 - ??? (uVar14 = 0)
1 - ??? (pcVar5 = (char *)0x1)

0x114 - Fly Up (Practice)

0x115 - Cloud Type
0 - Clouds
1 - Clouds (Higher) (Unused)
2 - Trees
3 - Clouds + Snowflakes

0x116 - Baxter Ready Animation
0 - Normal
1 - Slow
2 - Wind Up
