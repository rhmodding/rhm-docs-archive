Exhibition Match
---------------------
0x100 X - Input
X = Time in Ticks (Used Value: 0x20)

0x101 - Slugger Beat Animation
0x101<1> - Ready
0x101<2> X - Face after Pitch
0 - Blink 1 (Face Camera)
1 - Blink 2 ("Huh?")
2 - Blink 3 (Face Forward)

0x102 - Pitcher Animations
0 - Beat
1 - Ready
2 - Pitch
3 - Cap Off
4 - No Face Changes

0x103 - Monkey Animations
0 - Beat
1 - Throw

0x104 - Curtain
0 - Down
1 - Up
2 - Raise

0x105 X - Slugger Swing
0 - Normal
1 - Spin

0x106 X - Pitcher Face
0 - Normal
1 - Angry
0x106<1> X - Happy
0 - Enabled
1 - Disabled
0x106<2> - Set condvar to 1 if Slugger has hit enough baseballs

0x107 - VS
0 - Enable
1 - Disable

0x108 Y, Z - Zoom
X = Type
0 - Specific Coordinates
1 - Add to previous
Y = Zoom
Used Values: -0x64, 0xD8, 0xE4, 0xF0, 0xF4, 0x10C, 0x110, 0x11C, 0x128, 0x1400
Z = Time in Ticks
Used Values: 6, 0x18, 0x60, 0xC0

0x109 - Audience Applauds

0x10A X - Activate Spotlights
X = Time in Ticks (Used Value: 0x18)

0x10B - Flash Lights (visible when zoomed out)

0x10C - Sky
0 - Day (Practice)
1 - Night (Real Game)

0x10D - Audience
0 - Disabled (Practice)
1 - Enabled (Real Game)
---------------------
Subs:
0x56 - 4 Beat Animations
0x57 - 2 Beats, Ready Pitch
0x58 - Pitching
0x59 - Pitching (Blink 1)
0x5A - Pitching (Blink 2) (Unused)
0x5B - Pitching (Blink 3)
0x5C - 4 Rests (Effectively Nothing)
0x5D - VS
The following wait for 8 Beats before doing anything
0x5E - Zoom in 4 times
0x5F - Zoom in 3 times
0x60 - Zoom in Slowly
0x61 - Zoom in 2 times (Leftover)
0x62 - Zoom in Quickly, then slowly
0x63 - Zoom out very far
0x64 - Zoom out 4 times
0x65 - Zoom out 3 times (Unused)
0x66 - Zoom out Slowly
0x67 - Zoom out 2 times
0x68 - Zoom out Quickly, then slowly (Unused)
0x69 - Exhibition Match (Practice)
0x6A - Exhibition Match (Real Game)
0x6B - Exhibition Match (Skipped Practice)
