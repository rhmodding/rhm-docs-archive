Cheer Readers
---------------------
0x100 X, Y, Z - Input
X = Type
0 - Press A (White) (Unused)
1 - Press A (Black) (Unused)
2 - Hold B (Spin)
3 - Release B (Open)
4 - None (Unused)
5 - Press A (Flip)
Y = Time in Ticks (Used Values: 0x30, 0x3C, 0x60, 0x78)
Z = Effect
0 - Nothing
1 - "Yay!" (Solo)
2 - "Yay!" (Girls) (Unused)
3 - "Yay!" (All)
4 - "BOOM!"

0x101 X, Y - Flip Book
0x102 X, Y - Spin Book
0x103 X, Y - Open Book
Xth Column, Yth, Row
0x102<1> - All Flip (Unused)
0x102<1> - All Spin
0x103<1> - All Open

0x104 - Book Flipping SFX
X = Type
0 - Horizontal
1 - Vertical
2 - Diagonal

0x105 - Nothing (Unused)

0x106 X - Image
0 - panel_image_00 (Rhythm Tweezers/Fillbots)
1 - panel_image_01 (DJ School)
2 - panel_image_02 (Lockstep)

0x107 X - Beat Animation
0 - Normal
1 - Closed Books

0x108 0 - Books Gleam (Unused)

0x109 Y - Mouth Movement
X = Type
0 - Solo
1 - Girls
2 - All
Y = Lip Sync
0x61 - AO
0x65 - IE
0x69 - IE
0x6F - AO
0x75 - U

0x10A - Raise Books (Used in Practice)

0x10B - Nothing (Was Lights (Remix 9) in Fever)
