Flipper-Flop
---------------------
0x100 X, Y - Input
X = Time in Ticks (Used Value: 0x30)
Y = Type
0 - Press A
1 - Press B (Flipper Roll)

0x101 - Seals (Beat Animation)

0x102 X, Y - Raise Flipper
X = Type
0 - Right
1 - Left
Y = Speed
0 - Smooth
1 - Instant (Leftover)

0x103 - Seals (Flip)

0x104 X - Seals (Flipper Roll)
0 - Continuous rolls
1 - Last roll

0x105<0> - Captain Tuck (Steps Forward)
0x105<1> - Captain Tuck (Beat Animation)
0x105<2> X - Captain Tuck (Mouth Animation)
0 - "Attention, Company!"
1 - "That's it!"
0x105<3> X, Y - Captain Tuck (Flipper Roll Animation)
X = Beat Animation
0x50B2B1 - beat_just_00
0x50B2CB - beat_just_01
0x50B2E5 - beat_just_02
0x50B2FF - beat_miss_00
0x50B319 - beat_miss_01
Y = Face Animation
0 - No Face Change
0x50B2A4 - face_just_00
0x50B2BE - face_just_01
0x50B2D8 - face_just_02
0x50B2F2 - face_miss_00
0x50B30C - face_miss_01
0x105<4> - Already in Place (Remixes)

0x106 - Switch Captain Tuck's counting SFX
0 - Low (1~4) (Miss)
1 - Normal (1~4)
2 - High (5~10)

0x107 - Captain Tuck miss face(?)

0x108 - Captain Tuck counting SFX changing after the first four rolls(?)

0x109 X - Captain Tuck (Praise Animation)
X = Number of Random Choice Cases (Used value: 5)
case 0 - "Iizo!"/"Good job!"
case 1 - "Yosi!"/"Good!"
case 2 - "Sou!"/"Nice!"
case 3 - "Soda!"/"Well done!"
case 4 - "Sha!"/"Yes!"

0x10A X, Y - Sneaky Spirit (Barbershop Remix)
X = Appearence
0 - Disable (Unused)
1 - Enable
Y = Unknown
0 - Disable (Unused)
1 - Enable
