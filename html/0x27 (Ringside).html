Ringside
---------------------
0x100 X, Y - Input
X - Type
0 - Nod
1 - Flex, Step 1
2 - Flex, Step 2
3 - Pose
4 - Pose, Newspaper
Y - Time in ticks (Used Values: 0x60, 0x78, 0x90)

0x101 X - View
0 - Normal
1 - Zoomed Out

0x102 - Camera Men Bounce Up
0x102<1> X - Zoom
0 - Zoomed In
1 - Zoomed Out
0x102<2> X - Zoom Out (Beginning)
0 - Normal
1 - Unknown
0x102<3> - Practice
0x102<4> - Real Thing

0x103 X - Newspaper swirls over X ticks (Used Value: 0xC)
0x103<1> - Remove Newspaper

0x104 X, Y - Reporter
X - Mouth Movements
0 - "Wubbadubbadubba zat true?"
1 - "Whoa, you go big guy!"
2 - Speed Modifier(?)
Y - Time in Ticks (Used Values: 3, 4, 5, 6, 7)
0x104<1> - Raise Mic
0x104<2> - Lower Mic
0x104<3> - Initialize Blinking

0x105 1 - Camera Flash (Unused)

0x106 - Beat Animation

0x107 - Prepare for a pose

0x108 - Post-Pose Effect
0 - Enable
1 - Disable

0x109 X - Wrestler Sweat
0 - Disable
1 - Enable

0x10A X - Rhythm Arena
0 - Disable
1 - Enable
0x10A<1> - Beat Animation
0x10A<2> X, Y, Z - Arena zoom from X to Y over Z ticks
Used values (Beginning): 0x100, 0x180, 0xF0
Used values (Ending): 0x100, 0x100, 6
0x10A<3> X, Y, Z - Arena opacity from X to Y over Z ticks
Used values (Beginning): 0xFF, 0, 0xF0
Used values (Ending): 0xFF, 0xFF, 6
0x10A<4> - Dim Lights

0x10B X - "Live from the Rhythm Arena"
0x10B<1> X - "You're watching KRIQ"
0 - Disable
1 - Enable

0x10C - Background
0 - Practice
1 - Real Game
