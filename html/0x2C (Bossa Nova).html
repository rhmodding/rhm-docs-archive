Bossa Nova
---------------------
0x100 X, Y - Spawn Object
X = Time in Ticks (Used Value: 0x30)
Y = Object
0 - Ball
1 - Cube

0x101 - Cloud Dips Down
0x101<1> - Cloud Turns
0x101<2> Y, T - Cloud Movements
Y = End Y Position (Used Value: -0xAF)
T = Time (Used Value: 0x120)

0x102 - Your Starting Position
0 - Right
1 - Left

0x103 - Check voice variation, then sets condvar to it
0x103 Y = Voice Variants
X = Which
1 - Female
2 - Male
Y = Variant
0 - Miss
1 - Default
2 - Playful

0x104 - ???

0x105 Y - Background Movement (Instant)
0x105<1> Y, Z - Background Movement (Smooth)
Y = Target Y Position (Used Value: 0, but should be 0xBD8)
Z = Time in Ticks (Used Value: 0x2D00)

0x106 - Beetle Flies Away

0x107 - Worm Drops Down

0x108 - Lights on the House

0x109 X - Bug Light (Blue)
0 - Disable
1 - Enable

0x10A - Bug Light (Green) Scamper Away

0x10B 0xXXXXXX - Text (takes values from rvl.msbt)
0x50C4D4 - "rvlReceiveL_message_00" (Great bumping!)
0x50C4EB - "rvlReceiveL_message_01" (You can do it.)
0x50C502 - "rvlReceiveL_message_02" (Relax. Feel the rhythm.)
0x50C519 - "rvlReceiveL_message_03" (You don't miss a trick!)
0x50C530 - "rvlReceiveL_message_04" (Just a little longer...)
0x50C547 - "rvlReceiveL_message_05" (I need a new partner...)
0x50C55E - "rvlReceiveL_message_06" (You should go pro!)
0x50C575 - "rvlReceiveL_message_07" (We're a good team.)
0x50C58C - "rvlReceiveL_message_08" (We should just stop now.)
0x10B<1> - Remove Text

0x10C - ???
---------------------
Subs:
0x56 - Pattern (Inputs)
0x57 - Turn (Inputs)
0x58 - Voiceless Pattern
0x59 - Voiceless Turn
0x5A - Male Voice Clip ("Bounce, B-Bounce, B-")
0x5B - Male Voice Clip ("Bounce, Turn")
0x5C - Female Voice Clip ("-ce, Bounce, Boun-")
0x5D - Female Voice Clip ("-ce, Bounce, Spin")
These change the ratings category:
0x5E - Male Pattern ("Bounce, B-Bounce, B-")
0x5F - Male Turn ("Bounce, Turn")
0x60 - Female Pattern ("-ce, Bounce, Boun-")
0x61 - Female Turn ("-ce, Bounce, Spin")
These combine the pattern and voice clips (intended for remixes):
0x62 - Male Pattern ("Bounce, B-Bounce, B-") (Leftover)
0x63 - Male Turn ("Bounce, Turn") (Leftover)
0x64 - Female Pattern ("-ce, Bounce, Boun-") (Leftover)
0x65 - Female Turn ("-ce, Bounce, Spin") (Leftover)
0x66 - Bossa Nova (Real Game)
0x67 - Bossa Nova (Practice)
0x68 - Bossa Nova (Skipped Practice)
