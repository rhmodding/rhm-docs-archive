Launch Party
---------------------
0x100 X - Input
0 - 1 (Bowling Model)
1 - 3 (Family Model)
2 - 5 (Cracker Model)
3 - 7 (Bell Model)

0x101 X, Y - Counter (Unused)
X = Number (0~7)
Y = Color
0 - White (Bowling Model)
1 - Red (Family Model)
2 - Blue (Cracker Model)
3 - Green (Bell Model)

0x102 Y - Spawn Rocket
X = Which One
0 - Current
1 - Next
Y = Rocket Model
0 - Bowling Model
1 - Family Model
2 - Cracker Model
3 - Bell Model
4 - Don't Spawn (Unused)
5 - Don't Spawn

0x103 - Spawn Rocket

0x104 - Remove Rocket

0x105 - Background
0 - White Fades Away
1 - White Gone Instantly
2 - Background Moves Up
3 - ??? (Two Player Leftover)

0x106 X - Launch Pad Hover Motion
0x106<1> X, Y - Launch Pad Clockwise Hover (Two Player Leftover)
X = Action
0 - Disable
1 - Enable
Y = Time in Ticks (Used Values: 0x300, 0x600)

0x107 X - Star Amount
Used Values: 0xA, 0xF, 0x28, 0x64
0x107<1> X - Star Speed
Used Values: 0xA, 0x14, 0x1E, 0x28, 0x46, 0x64,
0xC8, 0xFA, 0x10E, 0x12C, 0x172, 0x190, 0x258
0x107<2> - Unknown
0x107<3> X - Size
Used Values: 0, 1, 0xA, 0x1E, 0x32, 0x3C, 0x50
0x107<4> Z - Z Axis Displacement
Used Values: 0x1E, 0x32, 0x64, 0x96, 0xFA, 0x12C, 0x190, 0x258
0x107<5> - Despawn Stars
0x107<6> X - Stars Respawn to the Top
0 - Disable
1 - Enable

0x108 X - Placement of Stars (Used Values: 1, 3)

0x109 X, Y, Z - Spawn Flying Rocket
T = Type
0 - 7 (Bell Model)
1 - 1 (Bowling Model)
2 - 3 (Family Model)
3 - 5 (Cracker Model)
X = X Position (Used Values: -0x69, -0x46, 0x50, 0x6E)
Y = Y Position (Used Values: -0xC, 0x17, 0x3A, 0x58)
Z = Time in Ticks (Used Value: 0x510)

0x10A X - Set Pitch (Used Values: 0, 0x18)
0x10A<1> - Update Pitch

0x10B - Launch Pad
0 - Normal
1 - Boost
2 - Big Boost

0x10C X - Unknown
0 - Disable (Unused)
1 - Enable

0x10D X, Y, Z - Launch Pad Z Movement (Two Player Leftover)
X = Which One
0 - Player 1
1 - Player 2 (Crashes)
Y = Z Axis (Used Values: 0xC0, 0x100, 0x180, 0x400)
Z = Time in Ticks (Used Values: 0xC0, 0x180)

0x10E X - Stars have Rings
0 - Disable
1 - Enable
---------------------
Subs:
0x56 - Bowling Model (Unused)
0x57 - Family Model (Unused)
0x58 - Cracker Model (Unused)
0x59 - Bell Model (Unused)
0x5A - Pitch Pattern
0x5B - Launch Party (Practice)
0x5C - Launch Party (Real Game)
0x5D - Launch Party (Skipped Practice)
