Board Meeting
---------------------
0x100 X, Y - Input
X = Time in Ticks (Used values: 0x24, 0x30, 0x48, 0x60, 0x90, 0x120)
Y = Type of Stop
0 - Individual
1 - Together

0x101 Y - Pig Animations (Individual)
0x102 - Pig Animations (All)
X = Animation
0 - Beat Animation
1 - Ready
2 - Start Spinning
3 - Stop Spinning
Y = Which one (0~3, left to right)

0x103 X - Beat Animation
0 - Disable
1 - Enable

0x104 X - Glasses Shine
0 - Disable
1 - Enable

0x105 - Assistant (Beat Animation)
0x105<1> X - Talking (X Times)
0x105<2> - Down (Unused)
0x105<3> - Up (Return w/ cups) (Unused)
0x105<4> - End (Talk w/ cups) (Unused)
0x105<5> - "One-two-three!"
---------------------
Subs:
0x56 - "One-two-three!" Voice
0x57 - "One-two-three!" Voice (Offbeat)
0x58 - Pigs Spin (Individual) (2.0 Beats) (Unused)
0x59 - Pigs Spin (Individual) (1.0 Beats)
0x5A - Pigs Spin (Individual) (0.5 Beats) (Leftover)
0x5B - Pigs Spin (Individual) (1/3 Beats) (Unused)
0x5C - Pigs Spin (Individual) (0.25 Beats) (Leftover)
0x5D - Pigs Spin (Together)
0x5E - Pigs Stop (Individual) (2.0 Beats)
0x5F - Pigs Stop (Individual) (1.0 Beats)
0x60 - Pigs Stop (Individual) (0.5 Beats) (Leftover)
0x61 - Pigs Stop (Individual) (1/3 Beats) (Unused)
0x62 - Pigs Stop (Individual) (0.25 Beats) (Leftover)
0x63 - Pigs Stop (Together)
0x64 - Pigs Stop (Together) (Offbeat)
0x65 - Board Meeting (Real Game)
0x66 - Board Meeting (Practice)
0x67 - Board Meeting (Skipped Practice)
