Monkey Watch
---------------------
0x100 X - Input
X = Time in Ticks (Used Values: 0x18, 0x30)

0x101 - Move Clock Hand to Next Position
0x101<1> X - Snap Clock Hand to Angle (Leftover from Remix 2) (360Â°)
0x101<2> X, Y, Z - Camera Movement
X = Move from Position (Used value: 0)
Y = Move to Position (Used value: 0)
Z = Time in Ticks (Used value: 0x30)
0x101<3> X, Y = Move Camera X degrees in Y ticks
X = Angle (Used Values: 0x18, 0x1E, 0x24)
Y = Time in Ticks (Used Value: 0x180)

0x102 X, Y - View
X = Type
0 - Zoomed in
1 - Zoomed out
Y = Speed
0 - Gradual
1 - Instant

0x103 X, Y - Doors
X = Position (0 = Top)
Y = Speed
0 - Gradual
1 - Instant
0x103<1> - ???
0x103<2> X - Opening Doors
0 - Disable (Leftover)
1 - Enable (Default) (Unused)

0x104 X - Monkey at X position play "Wait" animation (Used Value: 0)
0x104<1> - Monkey at 1 position from you plays "Wait" animation
0x104<2> - Monkey at 1 position from you plays "Ready" animation
0x104<3> X - Monkey at X positions from you will be purple

0x105 X - Remix 2 Intro Background (Leftover)
0 - Disable
1 - Enable

0x106 - Beat Animation (Watch)

0x107 - Spawn Balloon
0x107<1> - Remove Balloon
0x107<2> X - Snap to Position
X = X Position (Used Value: 0x180)
0x107<3> X, Y - Snap from this position (relative to last position) to last position
X = X Position (Used Value: -0x113)
Y = Y Position (Used Value: 0x64)
0x107<4> X, Y - Move balloon around
X = X Position (Used Values: -0x113, 0, 0x113)
Y = Radius (Used Values: -0x96, -0x4B, -0x32, 0, 0x64, 0x96, 0x113)

0x108 - Unknown (Unused)

0x109 - Unknown (Unused)

0x10A<1> 1, 1 - Related to having monkeys ready for high fives properly in remixes (Leftover)
