Blue Bear
---------------------
0x100 X, Y, Z - Input
X = Type
0 - Donut
1 - Cake
Y = Eating Animation
0 - Normal
1 - Long
Z = Nothing (Used Value: 0)

0x101 - Bear Animations
0 - Push left bag (Cake)
1 - Push right bag (Donut)
2 - Push both bags (Both)
3 - Open Mouth

0x102 X, Y - Memories
0x102<1> X - Fade out Memory
X = Slot
0 - Right Side
1 - Left Side
Y = Animation name
Used Values:
girl_in_R00
girl_in_R01
girl_in_L00
girl_in_L01
lostlove_inL
lostlove_inR

0x103 X, Y - Misc. Bear Animations
0x103<1> - Reset to default
0x103<2> X, Y - Misc. Bear Animations, then reset
X = Body Animation
Used Values:
0
all_intro_00
all_intro_01
all_breath
all_breathB
Y - Face Animation
Used Values:
0
face_smile
face_break
face_breakB
never_ending_smile

0x104 X - Crying
0 - Disable
1 - Enable

0x105 - Wind

0x106 - Dream Background Trigger

0x107 M, "X" - Opening Text (Left-Hand Remix)
0x107<1> - Remove Text
M = Which msbt file (Used value: 0xB)
0 - common
1 - game_info
2 - layout
3 - map
4 - coffee
5 - bonus
6 - treasure
7 - agb
8 - ntr
9 - rvl
0xA - ctr
0xB - remix
0xC - agb_result
0xD - ntr_result
0xE - rvl_result
0xF - ctr_result
0x10 - remix_result
0x11 - demo_dl
X = Text string
Used values:
"remix07_intro_a_00" (...Bearbara, my love.)
"remix07_intro_a_01" (I think I need to go on a diet.)
"remix07_intro_b_00" (I can't eat so many donuts.)
"remix07_intro_b_01" (I spent my whole allowance...)
