Catchy Tune
---------------------
0x100 X, Y, T, Z - Input
0x101 X, Y, T, U, Z - Input (Remix Transitions)
X = Type
0 - Orange (T = 0x30)
1 - Pineapple (T = 0x60)
2+- Pineapple (Broken) (Unused)
Y = Side
0 - Right
1 - Left
T = Time in Ticks until catch (Used values: 0x30, 0x60)
U = Time in Ticks until first bounce (Used values: 6, 0x36)
(If 0, the object bounces only twice, but still falls from the top, which looks awkward)
Z = Used if two are to be caught at once
0 - No
1 - Yes

0x102 X - Beat Animations
0 - Left
1 - Right

0x103 - Smile (if both are currently in the "catch" animation) Ignores 0x102 from now on

0x104 - Restore normal animation
---------------------
Subs:
0x56 - Nothing (Unused)
0x57 - Catchy Tune (Practice)
0x58 - Catchy Tune (Real Game)
0x59 - Catchy Tune 2 (Real Game)
0x5A - Catchy Tune (Skipped Practice)
