Sick Beats
---------------------
0x100 - Virus enters the pipe

0x101 X - Set next type of virus
0 - Blue
1 - Pink
2 - Black
3 - Invisible (Unused)

0x102 - Virus dashes to next position

0x103 X - Input
X = Time in Ticks (Used Value: 0x30)

0x104 - Nothing

0x105 - Sets condvar to 1 if the Virus is still alive

0x106 - Yellow thing
0 - Stay
1 - Eaten
2 - Pop
3 - Virus Laughs
4 - Heal

0x107 - Beat Animation

0x108 - Sets condvar to 1 if you Game Over

0x109 X - Set number of yellow things (lives)
X = Number (Used values: 1, 2, 3)

0x10A X - Setup
0 - Real Game
1 - Practice
2 - Final Remix
0x10A<1> - Sets condvar to 1 if playing in the Practice

0x10B - Save Score

0x10C - Yellow things will drop bonus items (Endless Game)
