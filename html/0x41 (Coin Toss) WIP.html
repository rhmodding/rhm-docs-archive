Coin Toss
---------------------
0x100 X - Input
Used Value: 0x30

0x101 - Hand Animation
0 - Reset(?)
1 - ???
2 - ???
3 - Open Hand (Unused)
4 - Disable Hand Out Animation (?)
5 - Open Hand Coin (Unused)

0x102 X - Set Progress
0x102<1> - Check Progress
0 - Before Flick
1 - Can Flick
2 - After Flick
3 - Caught the Coin
4 - Miss
5 - Game Clear
6 - Game Over

0x103 X - Watch
0 - Disable
1 - Enable

0x104 X - Tempo
Used Values: 0x50, 0x5A, 0x5F, 0x64, 0x69, 0x6E, 0x78

0x105 X, Y - Music
X = Variant (Used Values: 0, 1, 3, 4, 5, 0x14, 0x15, 0x16)
Y = Pitch (Used Values: 0, 1)

0x106 - ???
0 -
1 - Check game progress, set condvar to 0 if you miss

0x107 - Show Score Counter (Leftover, not the global counter)
0x107<1> - Add 1 Point
0x107<2> - Reset Score
0x107<3> - ??? Random?
0x107<4> - Save Score
0x107<5> X - Set Score

0x108 X - Lives
0 - Disable
1 - Enable
0x108<1> X - Set Amount
0x108<2> - ??? (Unused)
0x108<3> - Remove Life
0x108<4> - Check Amount

0x109 - Real Game (Removes timer, plays music instead of the tick-tocks, removes tutorial text)

0x10A - Final Remix
0 - Player Off-Screen
1 - Man On-Screen
2 - Player Slides In
3 - Man Flicks
4 - Man Slides Away
5 - Remix Illustration
6 - Remix Illustration Fades Out
