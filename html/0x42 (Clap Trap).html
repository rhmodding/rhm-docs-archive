Clap Trap
---------------------
0x100 X - Input
X = Time in Ticks (Used Value: 0x30, 0x60)

0x101 X - Setup
0 - Real Game
1 - Practice

0x102 X - Dim Lights
0 - Disable
1 - Enable

0x103 - Practice Speak (Leftover)

0x104 X - Breath
X = Time in Ticks during which it can happen (it's possible for it to never happen)
Used values: 0x18, 0x24, 0x30, 0x48, 0x60, 0x90, 0xC0, 0x120

0x105 - Face Reset
0 - Disable
1 - Enable (Normal)
2 - Enable (Game Over)

0x106 - Fall Over

0x107 - Sets condvar to 1 if player failed

0x108 X - Score Goal (Used Values: 0, 0xA, 0xF, 0x14)
0x108<1> - Sets condvar to 1 if player has achieved the score goal
0x108<2> - Save Score
