Introduction
---------------------
0x100 X, Y - Input
X = Time in Ticks (Used Value: 0x30)
Y = Type
0 - Normal
1 - Skill Star

0x101 - Free Mode
0 - Enable
1 - Disable

0x102 - Performance check
0 - Start performance check for 4 inputs. Freezes the game after the 4 inputs have been performed
1 - Resumes the game after freezing
2 - Checks if the player did 4 inputs, and sets the condvar to 1 if they did

0x103<0> X - Drummers Slide in
0x103<1> X - Drummer Beat Animation
0x103<2> X, Y - Ready to Hit
0x103<3> X, Y - Ready to Hit (Alt?)
0x103<4> X - Hit drum
0x103<5> X - Reset Face?
X = Which one
0 - Left
1 - Middle
2 - Right
Y = Which arm
0 - Right
1 - Left

0x104 - ???

0x105 - "Early!/Late!" indicator on the bottom screen
0 - Enable
1 - Disable

0x106 - Skill Star tutorial
0 - Make star appear
1 - Collect star
