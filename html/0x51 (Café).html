Café
---------------------
0x101 X, Y - Sets the position of the cursor
X = Page
0 - StreetPass Terrace
1 - Goat Stable
2 - Barista
3 - Shop
4 - Museum
Y = Option (Used values: 0, 1)
For example, 0x101 4, 1 sets to page 4 (Museum) and selects Talk to Curator
---------------------
Subs:
0x56 - "Go!" (when changing voice language)
0x57 - Redirects to page 2, option 0 (Talk to Barista/Talk to Customer)
0x58 - Redirects to page 0, option 0 (StreetPass Terrace)
0x59 - Redirects to page 1, option 0 (Feed Goat)
0x5A - Redirects to page 3, option 0 (Shop)
0x5B - Redirects to page 4, option 0 (To Museum)
0x5C - Redirects to page 4, option 1 (Talk to Curator)
All but 0x56 and 0x5A also play the Café music
